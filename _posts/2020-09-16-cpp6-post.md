### 복사 생성자와의 첫만남

---

### C++ 스타일의 초기화

```cpp

// C style
int num = 20;
int &ref = num;

// C++ style

int num(20);
int &ref(num);

```

```cpp

class SoSimple
{
    private :
        int num1;
        int num2;
    
    public :
        SoSimple(int n1, int n2) : num(n1), num2(n2)
        {}

        void ShowSimpleData()
        {
            cout<<num1<<endl;
            cout<<num2<<endl;
        }
};

int main(void)
{
    SoSimple sim(15,20);
    SoSimple sim2 = sim1;

    sim2.ShowSimpleData();
    return 0;

}

```

### 키워드 explicit


- ``SoSimple sim2 = sim1; `` -> ``SoSimple sim2(sim1)``

- 이러한 묵시적 형변환은 복사 생성자를 explicit 으로 선언하면 막을 수 있다.
