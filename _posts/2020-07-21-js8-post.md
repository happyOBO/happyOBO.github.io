### 개요

- Built-in이란?
    - 값 타입, 연산자, 오브젝트를 사전에 만들어 놓은것

- Built-in value type
    - Undefined, Null, Boolean, Number, String, Object
- Built-in Operatier
    - -, +, *, /, %, ++, etc

### Built-in object

- 자바스크립트의 기본적인 구조는 name/key 의 value로 구성되어 있는 property 이다.
- 총 11개의 object
- Number
    - 123과 같은 수자, 상수, 지수를 처리하는 오브젝트
- String
- Boolean
- Object
    - property 를 처리하기 위한 오브젝트
- Array
- Function
- date
- math
- JSON
    -서버와 데이터 송수신 지원
- RegExp
- global
    - 소스 파일 전체에서 하나만 존재
    - 예를 들어 넘버는 여러개 만들수 있지만, 이건 불가,,
    - 모든 코드에서 공유, 접근 가능

```javascript
var obj = Number;
obj.isNaN(123); \* false *\
```
