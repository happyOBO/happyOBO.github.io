### Xycar 개발 환경 구축

- 자이카 로그인 암호
    - 아이디 : `nvidia`
    - PW : `nvidia`

### 자이카에 무선 접속

- 무선 와이파이로 Xycar에 접속 가능
    - 차량을 주행시켜놓고 SW 개발 가능
    - 차량에 장착된 카메라에 찍히는 영상을 그 즉시 확인할 수 있어서 편리

- 리눅스는 원격 접속 기능 제공
    - 원격 터미널
    - 원격 데스크탑
    - 가상 네트워크 컴퓨팅

### 원격 접속

- 텍스트 기반
    - 노트북 / 스마트폰에서 Xycar로 SSH (TEXT 기반) 원격 접속
- GUI기반
    - 노트북 / 스마트 폰에서 Xycar 로 VNC 원격 접속
    - 안드로이드 : VNC Viewer
- TEXT 기반 원격접속을 위한 SSH 프로그램 - PUTTY
    - 윈도우, 리눅스 스마트폰 모두에서 가능
- 실습실에서 Xycar 리눅스에 접속
    - 와이파이 검색 / 접속
        - 무선 공유기는 `Xycar-A3-2201`
        - 비번은 `xycar123`
    - 리눅스 접속/ 로그인
        - VNC 앱 실행
        - 리눅스 IP 주소는 10.42.0.1
        - 로그인 아이디는 `nvidia`
        - 로그인 비번은 `nvidia`

### Xycar 와이파이에 접속하려면?

- 모형차에서 제공하는 와이파이 무선랜에 접속해야
    - Xycar가 제공하는 무선랜을 잡아야함
    - SSID 값을 알아야함 (SSID = 무선 공유기 이름) ~~자동차위에 장착된 프로세서 보드에 적혀있다~~
- 스마트폰으로 접속
    - 자이카 와이파이에 접속

### 리눅스 PC에서 터미널 사용해서 Xycar 원격 접속

- 터미널 열고

    ```bash
    ssh -Y nvidia@10.42.0.1
    nvidia
    ```
- 리눅스에서 기본적으로 Remmina가 기본 설치되어있다. VNC 설정후 10.42.0.1에 연결
