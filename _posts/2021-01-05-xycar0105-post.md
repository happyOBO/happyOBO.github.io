## 자이카 자율주행 기능 소개

### 자율주행 소프트웨어

- 규칙기반 주행 모델 + 학습 기반 주행 모델
    - 규칙 기반 접근 법 : 주행 과정 중 발생 가능한 다양한 상황을 정교하게 규칙으로 모델링
    - 학습 기반
        - 인간의 주행 과정 데이터화
        - 반복학습을 통해 스스로 터득

### 머신러닝 YOLO 활용 자율 주행

- 전반 상황을 보고 어떤 미션을 수행해야하는 지 파악
- 맵에 있을 오브젝트 : 3구 신호등, 4구 신호등, 회전 로터리, 언덕, 3차선 을 찾아서 라벨과 영역 표시


### 강화학습 기반 자율주행

- 가장 기본적인 강화학습 모델인 DQN 적용
- 시뮬레이터에서 벽과 충돌하지 않고 빠르게 주행하도록 만들기
    - 많은 횟수의 반복 주행 + 적절한 보상값 부여


### End-to-End 자율주행

- 정답을 같이 보여주면서 학습
- 영상과 핸들의 각도를 같이 저장


### Adaptive Cruise Control

- 추돌 방지 자율 주행
- 앞차랑 가까워지면 천천히 멀어지면 이동


### PID 제어

- 차량이 흔들거리면서 주행하지 않도록 제어

### AR Tag 기반의 주차

- AR Tag를 보고 자동차의 자세를 파악하여 똑바로 주차

### 직각 주차 / 평행 주차

- 빈 공간을 확인해서 주차 가능 여부 판단
- 신속하고 정확한 주차

### 라이다 기반 복도 주행

- 최대한 빠르게 주행

### 앞차 따라가기 주행

- 앞에서 달리는 차량이 지나간 경로를 따라서 주행
- 앞차의 이동 궤적 파악

### 비 보호 좌회전

-  규칙 기반 ? 학습 기반?

### 고속 코너링 - 드리프트 주행

- 고속으로 코너링 할때 바퀴를 반대방향으로 해서 코너를 고속으로 돌 수 있게끔

### 미션 수행에 필요한 기능 소개

- 차선 따라 주행
    - 카메라 영상을 기반으로 차선 인식
    - 차선을 벗어나지 않도록 주행
    - 영상에서 차선 찾기
        1. OpenCV 기반의 영상처리를 통해 전체 이미지에서 차선의 위치 파악
            - `Crayscale` 변환 -> 이진화 변환 -> 흰색이 많이 모여 있는곳을 차선으로 결정
        2. 허프변환을 이용한 차선 추출

            ![[no-alignment]]({{ site.url }}{{ site.baseurl }}/assets/ROS_lecture/0105/1.png){: .align-center}

        3. 머신러닝 이용
            - ENet : Real Time Semantic Segmentation
- 직전 ? 회전?
    - 양쪽 차선을 모두 찾을 필요는 없다.
    - 촬영된 사진을 얇게 슬라이스해서 화면 차선부분을 두 점으로 만들고 그 사이 중간 값이 화면과 일치하면 직진!
    - 벗어나있으면 그만큼 회전 시킨다.
- 정지선 인식
    - 화면 하단에 ROI 영역 설정
        - 하얀색 픽셀이 많으면 정지션으로 인식
        - 또는 수평선이면 정지선으로 인식

- 횡단보도 인식
    - OpenCV : 도형개수 카운트
        - 횡단보도 : 여러개의 블록 카운팅
        - 정지선 : 한개의 블록으로 카운팅
- 신호등 인식
    - 화면 상단에 ROI 영역 설정(난 위만 볼래!)
    - 연속된 동그라미 찾기
    - 또는 YOLO 오브젝트 인식
    - 신호등 상태 정보 획득
    - **색깔로 판단하지말고** 동그라미의 위치에서의 밝기로 파악한다.

- 언덕 구간
    - 차선 왜곡 -> 볼록한 항아리 모양으로 
    - 한쪽 차선만 보고가면 곡선 차선으로 오해하고 핸들 꺾을 ㅅ도
    - 양쪽 차선을 모두 봐야한다.
    - IMU 센서로 언덕 주행 인식
        - 천천히, 양쪽 차선을 모두 보고
- 장애물
    - 돌발장애물
        - 라이다를 이용하여 전방 장애물 감지
        - 0 ~360도, 1도 단위로 거리 센싱
        - (X,Y ) 좌표로 변환하여 장애물 위치 판단
- 로터리 진입
    - 라이다로 장애물 존재 여부 확인
        - 좌측에서 다가오는 차량 파악
        - 우측으로 멀어지는 차량 확인
    - 초록색 부분에 상대 차량이 있을 때 진입하자.

    ![[no-alignment]]({{ site.url }}{{ site.baseurl }}/assets/ROS_lecture/0105/2.png){: .align-center}

- 차선 변경
    - 자동차가 없는 차선으로 진행
    - 장애물 감지하여 확인
    - 3개 차선중 장애물이 없는 차선 선택 주행

- QR 코드
    - 현재 어떤 미션을 수해해야하는지 알려줌
    - 많은 기능의 동시구동은 힘드믈, 신호 표지판을 이용한다.
