## 자율 주행 자동차 기술 개요

### 자율주행 자동차는 4차 산업혁명의 아이콘?

- 자율주행 자동차 = 인공지능 + 빅데이터 + 초연결

### 자율주행 6단계 (미국 자동차 기술학회 SAE 6단계)

- 3단계는 마음을 놓고 있으면안된다. 경보가 울리면 잽싸게 핸들을 잡아야한다. 4단계는 잠을 자도 된다.

### 주변의 자율주행 자동차

- 현대차의 소나타
    - 승하차가 어려운 좁은 주차 공간에서 주차/ 출차를 도와주는 기술, 스마트 키를 통해 차량의 원격 시동과 전 / 후진 제어 가능
    - 후진 출차 시 후측방 사각지대에서 접근하는 차량 감지하여 운전자에게 경고 및 브레이크 제어
    - 후측방 충돌 방지 보조 : 아웃사이드미러로 확인이 어려운 후측방 사각지대에서 접근하는 차량을 감지, 경고 및 제동 제어를 통해 충돌 방지 보조
    - 전방 충돌 방지 보조 : 전방에 등장하는 자전거를 판단해 충돌 위험시 경고문 표시
    - 고속도로 주행 보조 : 고속도로 본선 주행시 전방 차량과의 거리, 차선 정보 내비게이션 정보를 이용해 차간 거리 제어 및 차로 유지를 보조하는 기능
    - 차로 유지 보조: 전방 카메라를 이용해 차선을 인식하고 차선의 가운데로 차량이 주행할 수 있도록 보조
- HDA : 고속도로 , LFA : 차로 유지 보조(Line Following Assist , 부분 자율 주행)
- 테슬라의 모델 3 : 2단계
    - 차선 변경 기능 : 방향 지시등을 키면 차선을 자동으로 변경
    - 오토 플롯 : 도로에서 차선을 그대로 유지하는 기능, 커브길에서는 회전
    - 안전 장치 : 차선이 겹치거나 다른 차에 충돌 위험이 있을 때, 차량이 자동으로 경고 및 제동
- 자율주행 택시
    - 구글 알파벳 자회사 웨이모

## 자율주행 단계별 요소 기술

### 자율주행 프로세스

- 자율 주행 3단계 프로세스는 인지, 판단, 제어로 구성된다.

    ![[no-alignment]]({{ site.url }}{{ site.baseurl }}/assets/ROS_lecture/0104/3.png){: .align-center}

### 사람의 운전 방식

- 네비게이션의 도움
    - 지도 제공
    - 목적지까지 경로 찾기
    - 좌회전 / 우회전 경로 변경 안내 제공

- 사람이 직접
    - 차선과 지형지물 보고 주행
    - 주변 차량 살펴서 주행
    - 교통 신호등 인식해서 주행
    - 돌발 상황 대응 

### 자율 주행 sw 가 운전하는 방식

- 보다 정확한 지도 필요
- 지도에서 자기의 현재 위치 알아내기 - GPS, 영상 매칭, 라이다 매칭
- 목적지까지 경로 찾기
- 주변 살피기 - 각종 센서
- 상황에 맞게 속도 조정, 핸들 꺾기
    - 경로 그대로 따라가기(차선 준수)
    - 교통 신호 따르기
    - 예외 상황에 대처 : 추월 , 정차
    - 돌발 상황에 대처 : 급 브레이크, 장애물 피해가기

### 고정밀 지도

- HD Map(High Def Map)
    - 기존 네비보다 정밀한지도 10 ~ 20 센티 정밀도 제공
- 다양한 부가정보 포함
    - 차선 정보
    - 가드레일
    - 도로 곡률 , 경사
    - 신호등 위치
    - 교통 표식

### Localization 기술

- 정밀지도와 연동하여 차량의 현재 위치 파악
    - 라이다, 카메라를 이용해 지도의 어느위치에 있는지?
    - 예 : 지도의 표지판과 라이다 /카메라로 인식한 표지판 비교해서 라이다와 카메라로 봤을 때의 표지판 거리와 지도와의 거리차이가 있으면 없애게끔 움직여야한다.


### Global Path Planning (Route Planning)

- 목적지까지의 경로 찾기
    - 중간 목적지 또는 최종 목적지 까지의 경로
    - 현재 네비에 다 있는 기능

    ![[no-alignment]]({{ site.url }}{{ site.baseurl }}/assets/ROS_lecture/0104/7.png){: .align-center}

### Object Detection

- 주변 차량 보행자 오토바이 자전거 으을 인식

### Object Tracking

- 각 오브젝트에 고유 ID를 부여하여 추적 **예상되는 주행 경로 예측**

    ![[no-alignment]]({{ site.url }}{{ site.baseurl }}/assets/ROS_lecture/0104/8.png){: .align-center}

    - 하늘위의 드론에서 계속적으로 정보를 주는건 어떨까

### Local Path Planning (Trajectory Planning)

- 다음 이동할 곳으로의 경로 찾기( 충돌 회피 고려)
    - 여러개의 후보 경로를 확보해야 후보를 여러개를 뽑아줘서 갑작스런 사건 대비
    - 끊임 없이 후보 삭제, 신규 후보 등록 작업을 반복
    - 실시간 성이 중요 -> 시스템 최적화 필요

### Behavior Selector

- 행위 결정
    - 운전 어떻게 할지 의사 결정
    - 운전 방법, 성향 : 누가 내쪽으로 끼어들고 있는데 옆 차선을 넘어서라도 비켜줘야할까?


### Local Paht Following (Trajectory Following)

- 제어 파트애 해당
- 경로를 따라 차량을 운전하기
- 사례 : Pure puruit 알고리즘
    - 차량의 현재 위치를 결정한다.
    - 차량에서 가장 가까운 경로상의 점을 찾는다.
    - 목표점을 찾는다.
    - 곡률을 계산하고 해당 곡률로 차량의 방향을 업데이트해준다.
    - 차량의 위치를 업데이트한다.

## 자율주행 자동차 구현 기술

### 자율주행 자동차 구현에 필요한 기술들

- 자율주행 알고리즘
    - 센싱
    - 인지
    - 의사결정
- 자율주행 클라이언트 시스템
    - 소프트웨어
    - 하드웨어
- 자율주행 클라우드 플랫폼
    - 분산 컴퓨팅
    - 분산 스토리지


### 자율주행 알고리즘

- 센싱 (sensing)
    - 주변 정보 획득
        - GPS
        - IMU
        - 라이다
        - 초음파
        - 카메라
- 인지 (perception)
    - Localization 자기 위치 파악
        - GPS 와 IMU 를 조합해서 위치 측정
        - Stereo 카메라 영상으로 위치 측정
        - 라이다 , 포인트 클라우드, 파티클 필터로 위치 측정
    - Objection Detection (인식)
        - CNN 딥러닝 기반의 인식 모델 사용
    - Object Tracking (추적)
        - 개체 이동궤적(trajectory)을 추적
        - 차량, 보행자와의 충돌 회피에 활용

- 의사결정 - decision(action paln)
    - 동작 예측
        - 다른 차량의 동작 예측
        - 확률 모델 만들고 확률 분포 구하기
    - 경로 계획
        - Cost function으로 최적경로 탐색
        - 계산량 줄이기 위해 확률 기법 적용
    - 장애물 회피 (충돌 방지)
        - 1단계 능동형, 충돌까지의 시간과 초소거리 추정치 뽑아서 경로를 다시 계획
        - 2단계 반응형, 이동경로상에 장애물이 감지되면 주행제어 시스템에 개입

### 자율주행 클라이언트 시스템

- 소프트웨어
    - 실시간성과 신뢰성 확보 필요
    - ROS 문제점 해결 필요
        - Master가 죽으면 전체 시스템이 다운되며 복구용도의 모니터가 없음. 대책 필요
        - 메시지를 브로드 캐스팅 하면서 성능이 저하됨. 멀티 캐스팅 매커니즘 적용하면 좋음
        - 노드가 해킹되기 쉬움. 리눅스 컨테이너 : 샌드박스로 보안 기능 강화 가능
        - 로컬에서는 TCP/IP 통신 대신에 공유메모리 통신방식으로 적용하면 좋음
- 하드웨어
    - 성능향상필요
    - 파이프라인 병렬 프로세싱 기능 필요
    - HW 가성비 좋게 만들어야
    - 차량의 배터리 문제가 있으므로 전력 소모량 최소화 노력 필요
    - 차량이라는 환경에서는 발열 문제가 심각, 발열을 최소화 하거나 열을 쉽게 배출 시킬수 있는 방법을 찾아야한다.
