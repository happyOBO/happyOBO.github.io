## 리눅스 tar.gz 파일 설치

**[참고 링크](https://sungsunc.tistory.com/116)**

### 압축풀기

```
gzip -d filename.tar.gz
```

### 압축 푼것 설치하기

```
tar xvf filename.tar
```

### 한번에 압출 풀기 및 설치

```
tar zxvf filename.tar.gz
```

## 리눅스 tar.xz 파일 설치

```
tar xf filename.tar.xz
```
